# Mobile TPM Team Wiki Mockup

A modern, interactive mockup of the Mobile Technical Program Manager (TPM) team's internal wiki page at Uber.

## Overview

This project recreates the team's internal documentation page with a clean, responsive design that maintains the original structure while adding modern UI enhancements.

## Features

- **Team Mission Statement** - Highlighted mission in an eye-catching banner
- **Important Release Dates** - Key deadlines and schedules for mobile releases
- **Team Directory** - Interactive grid showcasing all team members with their roles and locations
- **Contact Information** - Slack channels, email, on-call schedules, and intake process
- **Activity Feed** - Real-time updates and changes to the wiki
- **Interactive Elements** - Like button, comments section, and hover effects

## Structure

```
mobile-tpm-wiki/
├── index.html      # Main HTML structure
├── styles.css      # Complete styling
├── script.js       # Interactive features
└── README.md       # This file
```

## Technologies

- Pure HTML5
- CSS3 with Grid and Flexbox
- Vanilla JavaScript
- Responsive design for all screen sizes

## Local Development

Simply open `index.html` in your browser, or run a local server:

```bash
python3 -m http.server 8000
```

Then visit: `http://localhost:8000`
